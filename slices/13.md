## 13. Regular Expressions

### ✅ Syntax & Creating Regex

A **regular expression** is a pattern used to match character combinations in strings.

#### Two ways to create

```js
// 1. Literal syntax (preferred — compiled at load time)
const re = /hello/;

// 2. Constructor (useful when pattern is dynamic)
const pattern = "hello";
const re2 = new RegExp(pattern);
const re3 = new RegExp("hello", "i"); // with flags
```

Use the constructor when the pattern comes from a variable or user input.


### ✅ Flags

Flags modify how the regex engine behaves. They go after the closing `/`.

| Flag | Name | What it does |
|------|------|-------------|
| `g` | Global | Find **all** matches, not just the first |
| `i` | Case-insensitive | `A` matches `a` |
| `m` | Multiline | `^` and `$` match start/end of **each line** |
| `s` | Dotall | `.` matches **newlines** too |
| `u` | Unicode | Enables full Unicode support |
| `y` | Sticky | Match only at the current `lastIndex` position |

```js
const str = "Hello hello HELLO";

console.log(str.match(/hello/));   // ['hello'] — first match only
console.log(str.match(/hello/gi)); // ['Hello', 'hello', 'HELLO'] — all, case-insensitive
```

#### Multiline flag

```js
const text = "start\nmiddle\nend";

console.log(text.match(/^\w+/g));  // ['start'] — without m
console.log(text.match(/^\w+/gm)); // ['start', 'middle', 'end'] — with m
```


### ✅ Character Classes

Character classes match a **set** or **type** of characters.

#### Shorthand classes

| Pattern | Matches |
|---------|---------|
| `.` | Any character except newline (use `s` flag for newlines) |
| `\d` | Digit `[0-9]` |
| `\D` | Non-digit `[^0-9]` |
| `\w` | Word character `[a-zA-Z0-9_]` |
| `\W` | Non-word character |
| `\s` | Whitespace (space, tab, newline) |
| `\S` | Non-whitespace |

#### Custom sets

```js
/[abc]/      // matches a, b, or c
/[a-z]/      // any lowercase letter
/[A-Z]/      // any uppercase letter
/[0-9]/      // any digit (same as \d)
/[^abc]/     // anything except a, b, c (negation)
/[a-zA-Z]/   // any letter
```

```js
console.log("cat".match(/[cbr]at/)); // ['cat']
console.log("bat".match(/[cbr]at/)); // ['bat']
console.log("rat".match(/[cbr]at/)); // ['rat']
console.log("fat".match(/[cbr]at/)); // null
```


### ✅ Quantifiers — Greedy vs Lazy

Quantifiers specify **how many times** a character or group can repeat.

#### Quantifier reference

| Quantifier | Meaning |
|-----------|---------|
| `*` | 0 or more |
| `+` | 1 or more |
| `?` | 0 or 1 (optional) |
| `{n}` | Exactly n times |
| `{n,}` | n or more times |
| `{n,m}` | Between n and m times |

```js
/\d+/    // one or more digits
/\d{3}/  // exactly 3 digits
/\d{2,4}/ // 2 to 4 digits
/colou?r/  // matches 'color' and 'colour'
```

#### Greedy vs Lazy

By default, quantifiers are **greedy** — they match **as much as possible**.
Add `?` after a quantifier to make it **lazy** — matches **as little as possible**.

```js
const html = "<div>hello</div>";

// Greedy — matches from first < to last >
console.log(html.match(/<.+>/)[0]);  // '<div>hello</div>'

// Lazy — matches the shortest possible string
console.log(html.match(/<.+?>/)[0]); // '<div>'
```


### ✅ Anchors & Boundaries

Anchors don't match characters — they match **positions** in the string.

#### Position anchors

| Pattern | Matches position |
|---------|----------------|
| `^` | Start of string (or line with `m` flag) |
| `$` | End of string (or line with `m` flag) |
| `\b` | Word boundary (between `\w` and `\W`) |
| `\B` | Non-word boundary |

```js
/^hello/   // string must start with 'hello'
/world$/   // string must end with 'world'
/^hello world$/ // exact full match
```

```js
console.log(/^hello/.test("hello world")); // true
console.log(/^hello/.test("say hello"));   // false

console.log(/world$/.test("hello world")); // true
console.log(/world$/.test("world peace")); // false
```

#### Word boundaries `\b`

```js
const str = "cat concatenate cats";

console.log(str.match(/\bcat\b/g));  // ['cat'] — exact word 'cat' only
console.log(str.match(/cat/g));      // ['cat', 'cat', 'cat'] — anywhere
```

`\b` matches where a word character meets a non-word character (or start/end of string).


### ✅ Groups

#### Capturing groups `()`

Wrap a part of the pattern in `()` to **capture** it — the match is stored and can be referenced.

```js
const date = "2024-06-15";
const match = date.match(/(\d{4})-(\d{2})-(\d{2})/);

console.log(match[0]); // '2024-06-15' — full match
console.log(match[1]); // '2024'        — group 1
console.log(match[2]); // '06'          — group 2
console.log(match[3]); // '15'          — group 3
```

#### Non-capturing groups `(?:)`

Group without storing the match — useful when you want to apply a quantifier to a group but don't need to capture it.

```js
// Capturing — stores 'ha' repeated
console.log("hahaha".match(/(ha)+/));  // ['hahaha', 'ha', ...]

// Non-capturing — groups without storing
console.log("hahaha".match(/(?:ha)+/)); // ['hahaha'] — no extra group
```

#### Named groups `(?<name>)`

Give a capture group a name for clearer access.

```js
const date = "2024-06-15";
const match = date.match(/(?<year>\d{4})-(?<month>\d{2})-(?<day>\d{2})/);

console.log(match.groups.year);  // '2024'
console.log(match.groups.month); // '06'
console.log(match.groups.day);   // '15'
```


### ✅ Lookahead & Lookbehind

**Lookarounds** match a position based on what's around it, **without consuming characters**.

#### Positive Lookahead `(?=...)`

Match something **only if followed by** a pattern.

```js
// Match digits only if followed by 'px'
console.log("14px 20em 10px".match(/\d+(?=px)/g)); // ['14', '10']
```

#### Negative Lookahead `(?!...)`

Match something **only if NOT followed by** a pattern.

```js
// Match digits NOT followed by 'px'
console.log("14px 20em 10px".match(/\d+(?!px)\b/g)); // ['20']
```

#### Positive Lookbehind `(?<=...)`

Match something **only if preceded by** a pattern.

```js
// Match digits only if preceded by '$'
console.log("$100 £200 $300".match(/(?<=\$)\d+/g)); // ['100', '300']
```

#### Negative Lookbehind `(?<!...)`

Match something **only if NOT preceded by** a pattern.

```js
// Match digits NOT preceded by '$'
console.log("$100 £200 $300".match(/(?<!\$)\d+/g)); // ['200']
```

#### Summary table

| Pattern | Reads as |
|---------|---------|
| `x(?=y)` | `x` followed by `y` |
| `x(?!y)` | `x` NOT followed by `y` |
| `(?<=y)x` | `x` preceded by `y` |
| `(?<!y)x` | `x` NOT preceded by `y` |


### ✅ Backreferences

A backreference refers to a **previously captured group** in the same pattern.
Use `\1`, `\2`... for numbered groups, or `\k<name>` for named groups.

```js
// Match repeated words
const re = /\b(\w+)\s+\1\b/;

console.log(re.test("hello hello")); // true
console.log(re.test("hello world")); // false
```

#### Named backreference

```js
const re = /(?<word>\w+)\s+\k<word>/;

console.log(re.test("bye bye"));   // true
console.log(re.test("bye hello")); // false
```


### ✅ Methods

#### `RegExp.prototype.test(string)`

Returns `true` or `false` — just checks if there's a match.

```js
const re = /^\d{5}$/;
console.log(re.test("12345")); // true
console.log(re.test("1234"));  // false
```

#### `String.prototype.match(regex)`

Returns an array of matches (or `null` if no match).

```js
const str = "cat bat sat";

// Without g — returns first match + groups
console.log(str.match(/[a-z]at/));  // ['cat', index: 0, ...]

// With g — returns all matches, no groups
console.log(str.match(/[a-z]at/g)); // ['cat', 'bat', 'sat']
```

#### `String.prototype.matchAll(regex)`

Returns an iterator of **all matches** including capture groups. Requires `g` flag.

```js
const str = "2024-01-01 and 2025-06-15";
const re = /(\d{4})-(\d{2})-(\d{2})/g;

for (const match of str.matchAll(re)) {
  console.log(match[0], match[1], match[2], match[3]);
}
// 2024-01-01  2024  01  01
// 2025-06-15  2025  06  15
```

#### `String.prototype.search(regex)`

Returns the **index** of the first match, or `-1` if not found.

```js
console.log("hello world".search(/world/)); // 6
console.log("hello world".search(/xyz/));   // -1
```

#### `String.prototype.replace(regex, replacement)`

Replaces matches with a string or function.

```js
// Simple replace
console.log("hello world".replace(/world/, "JS")); // 'hello JS'

// Replace all with g flag
console.log("aabbcc".replace(/b/g, "X")); // 'aaXXcc'

// Using capture groups in replacement ($1, $2...)
const date = "2024-06-15";
console.log(date.replace(/(\d{4})-(\d{2})-(\d{2})/, "$3/$2/$1")); // '15/06/2024'

// Using named groups
console.log(
  date.replace(
    /(?<year>\d{4})-(?<month>\d{2})-(?<day>\d{2})/,
    "$<day>/$<month>/$<year>"
  )
); // '15/06/2024'

// Using a function
console.log("hello world".replace(/\w+/g, (word) => word.toUpperCase()));
// 'HELLO WORLD'
```

#### `String.prototype.replaceAll(string|regex, replacement)`

Replaces all occurrences. With regex, must use `g` flag.

```js
console.log("aabbcc".replaceAll("b", "X")); // 'aaXXcc'
```

#### `String.prototype.split(regex)`

Splits a string using a regex as the delimiter.

```js
console.log("one1two2three".split(/\d/));    // ['one', 'two', 'three']
console.log("a,  b,   c".split(/,\s*/));     // ['a', 'b', 'c']
```


### ✅ Common Patterns

```js
// Email (basic)
/^[^\s@]+@[^\s@]+\.[^\s@]+$/

// URL
/https?:\/\/(www\.)?[\w-]+(\.[\w-]+)+(\/[\w-./?%&=]*)?/

// Phone (BD format: 01XXXXXXXXX)
/^01[3-9]\d{8}$/

// Positive integer
/^\d+$/

// Decimal number
/^\d+(\.\d+)?$/

// Hex color
/^#([a-fA-F0-9]{6}|[a-fA-F0-9]{3})$/

// Trim extra spaces
str.replace(/\s+/g, " ").trim()

// Only letters
/^[a-zA-Z]+$/

// Slug (url-safe)
/^[a-z0-9]+(?:-[a-z0-9]+)*$/

// Strong password (min 8 chars, upper, lower, digit, special)
/^(?=.*[a-z])(?=.*[A-Z])(?=.*\d)(?=.*[\W_]).{8,}$/
```

#### Using patterns in practice

```js
function isValidEmail(email) {
  return /^[^\s@]+@[^\s@]+\.[^\s@]+$/.test(email);
}

console.log(isValidEmail("atik@example.com")); // true
console.log(isValidEmail("not-an-email"));     // false

function slugify(text) {
  return text
    .toLowerCase()
    .replace(/\s+/g, "-")       // spaces → hyphens
    .replace(/[^\w-]/g, "")    // remove non-word chars
    .replace(/--+/g, "-");     // collapse multiple hyphens
}

console.log(slugify("Hello World! 2024")); // 'hello-world-2024'
```
